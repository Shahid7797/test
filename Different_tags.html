Types of Tags

Block Level Tags

In-Line Tags

Block level Tags
if we write the multiple HTML tags , the output will be reflected line - by - line. Eg: Headings , hr , br
In-line Tags
If we write the different HTML Tags, the output will reflect in the same - line. Eg: Strong, b, u,strike , i etc

Other types of Tags

Paired Tags

Un-Paired Tags

Paired Tags
Tags for which consists both the opening tag and closed tag. Eg: h1 , h2 , h3 , strong , b etc
Un-Paired Tags
Tags which consists only the opening tag. eg : hr , br,
